# Tiva-C Launchpad & Arduino UNO I2C Example
A simple example to test I2C module of Tiva-C Launchpad TM4C123GH6PM. An Arduino UNO board is used to test with it.

## Pins Connection
Tiva-C Launchpad       Arduino UNO
I2C3SCL (PD0)     <-->   SCL
I2C3SDA (PD1)     <-->   SDA
